# Smart_Expense_Tracker

Author: Nazima Praveen

Description:
This project is a desktop application built using Java Swing, JDBC, and MySQL.
It helps users track their daily expenses efficiently.

Features:
- User Registration and Login
- Add, View, and Delete Expenses
- Stores data in MySQL database

Technologies Used:
- Core Java
- Swing (GUI)
- JDBC
- MySQL

How to Run:
1. Create database using provided SQL
2. Update DB credentials in DBConnection.java
3. Run Main.java
